# Nimble Challenge

Este proyecto es un mini-challenge en **React + Vite** que consume una API pública para:

- **Step 2:** obtener datos del candidato por email.
- **Step 3:** obtener la lista de posiciones disponibles (jobs).

> Más adelante se completa el flujo de postulación por cada posición (Step 4/5).

---

## Tech stack

- React
- Vite
- Fetch API
- Variables de entorno con `.env.local`
